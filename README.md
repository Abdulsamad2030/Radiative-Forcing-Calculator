# Radiative Forcing Calculator

A small interactive tool that visualizes how CO₂ concentration drives radiative forcing and estimated equilibrium warming, based on the Myhre et al. (1998) logarithmic model:

```
ΔF = 5.35 × ln(C / C₀)
ΔT = ΔF / λ
```

Adjust the sliders to explore:
- Current CO₂ concentration (ppm)
- Pre-industrial baseline (C₀)
- Climate feedback parameter (λ)

Built as a companion piece to my B.Sc. Physics thesis, *"Radiative Forcing and Climate Sensitivity: Evaluating the Role of Greenhouse Gases in the Recent Climate Trends."*

No build step required — it's a single static HTML file, ready for GitHub Pages.
